# FIAP Metabase
This is a shared metabase to run locally so we can share dashboards

The Metabase metadata DB data should be stored at **postgres_data**

## Usage Instructions
First run the command below to run the docker containers (they take a while to start-up go get a coffee or something)
```bash
docker-compose up -d
```

Access at [localhost](http://localhost:3000/)
